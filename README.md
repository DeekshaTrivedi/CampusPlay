CampusPlay is a full-stack MERN web application developed to improve real-world interaction among college students through sports and activity coordination. 
The platform allows students to create and join game sessions using a poll-based system while preventing scheduling conflicts through smart participation logic. 
It also enables students of the same college to connect with peers having specific skill sets for projects, teamwork, and collaboration. 
The project demonstrates backend API development, authentication, database management, and responsive frontend design.
